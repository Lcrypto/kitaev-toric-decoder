

        Vladimir Kolmogorov. "Blossom V: A new implementation of a minimum cost perfect matching algorithm."
        In Mathematical Programming Computation (MPC), July 2009, 1(1):43-67.
        
------------------------------------------------------------------------------------------------------------------

To run the simulation, this folder needs to contain BLOSSOM-V decoder files.

Download the appropriate zip file from:	 http://pub.ist.ac.at/~vnk/software.html 
and extract it in this folder. 

See the output of "tree" below to get an idea of intended directory structure.
